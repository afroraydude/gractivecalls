Welcome to the Active Calls API.

Please note, all requests should be GET requests. Any other request type is not supported and will return a 404.
Usage:

  • /api/active - returns a list of active calls based on data from upstream
  • /api/calls - returns a list of the most recent calls in the system. Presented in JSON.
    Optional parameters:
    • ?count=<number> - returns the specified number of calls (defaults to 10, max 100)
    • ?fromDate=<date> - returns calls during or after the specified date (defaults to UNIX epoch)
    • ?toDate=<date> - returns calls before or during the specified date (defaults to current date)
    • ?district=<district> - returns calls for the specified district (defaults to all districts)
    • ?agency=<agency> - returns calls for the specified agency (defaults to all agencies)
  • /api/calls.csv - returns a CSV of the most recent calls in the system.
    Optional parameters:
    • ?count=<number> - returns the specified number of calls (defaults to 10, max 100)
    • ?fromDate=<date> - returns calls during or after the specified date (defaults to UNIX epoch)
    • ?toDate=<date> - returns calls before or during the specified date (defaults to current date)
    • ?district=<district> - returns calls for the specified district (defaults to all districts)
    • ?agency=<agency> - returns calls for the specified agency (defaults to all agencies)
  • /api/since?minutes=<number> - returns a list of calls since the specified number of minutes ago. Presented in JSON.
  • /api/today - returns a list of calls for today. Presented in JSON.
  • /api/today.csv - returns a CSV of calls for today.
